# [Selenized](https://github.com/jan-warchol/selenized) for [Zed](https://zed.dev/)

> 🌓 Selenized color palette — warm aesthetics for machines and people

Four themes: **Selenized Dark**, **Selenized Black**, **Selenized Light**, and **Selenized White**.

Based on [Jan Warchoł's Selenized palette](https://github.com/jan-warchol/selenized) — LAB-canonical colors computed from perceptual color models.

## Develop

- Make changes to `selenized.py` as you see fit
- Then run it with `make` (or `python3 selenized.py`) to generate `themes/selenized.json`
- Don't forget to increment the version number in `extension.toml` before committing
- Then update [the `extensions` repo](https://github.com/zed-industries/extensions) (see [docs](https://zed.dev/docs/extensions/developing-extensions#updating-an-extension))
